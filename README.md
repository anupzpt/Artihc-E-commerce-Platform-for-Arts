# Artihc-E-commerce-Platform-for-Arts 🎨✨

**Artihc-E-commerce-Platform-for-Arts** is an e-commerce platform designed for buying and selling unique artworks, built using **PHP, Laravel, and MySQL**. Developed as a major academic project during my **5th Semester**, the platform offers a seamless shopping experience with features like automated email confirmations, real-time stock management, and role-based access control. It provides tailored interfaces for both art enthusiasts and administrators, ensuring smooth navigation, efficient order handling, and secure account management.

This project showcases the practical application of software engineering principles and Laravel's powerful **MVC framework** to build a feature-rich, scalable e-commerce solution. It emphasizes clean code, database integration, user authentication, and responsive design.

---

## 🚀 Key Features 🖼️

* **Art Marketplace:** Seamless browsing, buying, and selling environments for artists and collectors.
* **Role-Based Access Control (RBAC):** Dedicated interfaces and permissions tailored for both standard users and platform administrators.
* **Real-Time Stock Management:** Dynamic inventory tracking to ensure artwork availability is accurate down to the second.
* **Automated Confirmations:** Instant email notifications dispatched to users upon order placements and account updates.
* **Order Handling:** Simplified administrative pipeline for tracking, updating, and managing customer transactions efficiently.

---

## 🛠️ Tech Stack & Tools 🔧

* **Backend Framework:** PHP (Laravel MVC)
* **Database:** MySQL
* **Frontend UI:** Blade Templating, HTML5, CSS3, JavaScript (Responsive Design) 
* **Authentication:** Laravel Auth / Breeze
