Artihc is an online art marketplace built using PHP, Laravel, and MySQL. Designed for buying and selling artworks, the platform offers a seamless shopping experience with features like automated email confirmations, real-time stock management, and role-based access control. It provides tailored interfaces for both users and administrators, ensuring smooth navigation, efficient order handling, and secure account management.This project showcases the use of Laravel's powerful MVC framework to build a feature-rich, scalable e-commerce solution. Artihc emphasizes clean code, database integration, user authentication, and responsive design
